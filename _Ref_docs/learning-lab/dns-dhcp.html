DNS & DHCP Basics

Duration: 25-40 minutes

Domain Name System (DNS)

What is DNS?

DNS is like the internet's phone book. It translates human-readable domain names (like google.com) into IP addresses that computers can understand.

DNS Servers

Different types of DNS servers work together to resolve domain names: Root servers, TLD servers, and Authoritative servers.

DNS Cache

Your device and local DNS servers keep a temporary record of resolved domain names to speed up future requests.

Dynamic Host Configuration Protocol (DHCP)

What is DHCP?

DHCP automatically assigns IP addresses and network configuration to devices when they join a network.

The DORA Process

DHCP uses a four-step process: Discover, Offer, Request, and Acknowledge (DORA) to assign IP addresses.

Lease Time

IP addresses are assigned for a specific time period (lease). Devices must renew their lease to keep using the same IP.

Hands-On Practice

Try these commands to explore DNS and DHCP on your computer:

  • nslookup google.com - Look up DNS information for a domain
  • ipconfig /all - View DHCP lease information
  • ipconfig /displaydns - View your DNS cache
