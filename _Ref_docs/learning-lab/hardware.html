CPU & RAM Basics

Duration: 65-80 minutes

Central Processing Unit (CPU)

What is a CPU?

The CPU is the brain of your computer, executing instructions and performing calculations for all programs and operations.

Clock Speed

Measured in GHz (gigahertz), clock speed indicates how many cycles the CPU can execute per second.

Cores

Modern CPUs have multiple cores, allowing them to process multiple tasks simultaneously.

CPU Components

Control Unit

Manages and coordinates CPU operations, directing the flow of data between CPU components.

ALU

The Arithmetic Logic Unit performs mathematical calculations and logical operations.

Cache

High-speed memory built into the CPU for quick access to frequently used data.

Random Access Memory (RAM)

What is RAM?

RAM is your computer's short-term memory, storing data that active programs need for quick access.

Volatile Memory

RAM is volatile, meaning it loses its contents when power is turned off, unlike storage devices.

Memory Speed

RAM speed is measured in MHz and affects how quickly data can be read or written.

RAM Types and Usage

DDR4

Current standard for RAM, offering high speeds and improved power efficiency.

Memory Channels

Multiple channels allow parallel data transfer between RAM and CPU for better performance.

Virtual Memory

When RAM is full, the system uses hard drive space as virtual memory, though it's much slower.

Hands-On Practice

Try these commands to view your system's hardware information:

  • wmic cpu get name - View CPU model
  • wmic memorychip get capacity - View RAM capacity
  • systeminfo - View complete system information
